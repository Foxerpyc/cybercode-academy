CyberCode Academy
https://foxerpyc.github.io/cybercode-academy/

CyberCode Academy is a web-based interactive HTML learning system designed for beginners.  
The application focuses on helping users understand basic HTML structure through guided practice, live preview, and test-based evaluation.

---

Project Information

- **Course**: TTTH 6214  
- **Project Type**: Individual Final Project  
- **Domain**: Education – Online Learning Platform  
- **Technology Stack**: HTML, CSS, JavaScript  

---

Project Purpose

The purpose of CyberCode Academy is to support beginner-level HTML learning by providing:

- Immediate visual feedback through live preview
- Non-intrusive learning hints during code input
- A controlled evaluation mechanism through a test button
- A simple progress system to encourage learning motivation

---

Key Features

### 1. Live HTML Preview
- Instantly displays the visual output of user-written HTML code
- Helps users understand how code affects layout and content

### 2. Learning Hint System
- Provides guidance while users type
- Does not evaluate or interrupt the learning process
- Designed to reduce cognitive overload

### 3. Test-Based Evaluation
- Code evaluation occurs only when the **TEST** button is clicked
- Checks for basic HTML elements such as `<h1>`, `<p>`, and `<html>`
- Displays structured feedback in the test terminal

### 4. Progress and XP System
- Users earn experience points (XP) after each test
- Progress bar visually represents learning advancement
- Encourages continuous improvement

---

How to Use

1. Open the website using a web browser
2. Enter HTML code in the editor panel
3. Observe changes in the live preview panel
4. Read learning hints displayed below the editor
5. Click the **TEST** button to evaluate your HTML
6. Review feedback and score in the terminal
7. Improve the code and repeat the test if necessary
